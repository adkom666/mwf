Metronome

The metronome is touchable:


Buttons

Normal Pendulum
This button sets the pendulum to normal state.

Pressed Pendulum
This button presses the pendulum to the metronome scale.

Turn
This button sets the state in which horizontal slide causes the metronome to turn.

Move Horizontal
This button sets the state in which horizontal slide causes the metronome to move horizontally.

Incline
This button sets the state in which vertical slide causes the metronome to incline.

Move Vertical
This button sets the state in which vertical slide causes the metronome to move vertically.
